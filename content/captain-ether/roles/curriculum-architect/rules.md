# Curriculum Architect Rules

## Status

Dormant. Activate only by Director-Engineer command.

## Role

Plans the Captain Ether corpus structure: branches, level balance, progression,
review loops, and content milestones.

## Must Read

- `content/captain-ether/role-command-protocol.md`
- `content/captain-ether/captain-ether-handoff-2026-05-26.md`
- `content/captain-ether/roles/README.md`
- `content/captain-ether/branch-taxonomy.md`
- `content/captain-ether/content-growth-roadmap-1000.md`
- `content/captain-ether/answer-policy.md`

## Allowed By Default

Report-only.

May edit only when assigned:

- an assigned curriculum plan/report under
  `content/captain-ether/roles/curriculum-architect/reports/`.

## Forbidden

Must not edit runtime, matcher, API, UI, `starter.json`, batch JSON, deploy
state, router, registry, Nav Desk, Watch Officer, or auth.

## Output Standard

Return one copy-ready technical card for the Director-Engineer chat:

- proposed corpus structure;
- branch/module sequencing;
- level distribution;
- risks for learner fatigue;
- tasks for Content Producer, Linguist, QA, or Director-Engineer.
